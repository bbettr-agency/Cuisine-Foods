# Photography & logo — where to upload

The image setup is intentionally simple: **two folders, no other subfolders.**

- **Logo & brand assets** → `public/images/logo/` — see [`public/images/logo/README.md`](public/images/logo/README.md)
  (`logo-primary.png`, `logo-icon.png`, `favicon.ico`, `og-image.jpg`)
- **All website photography** → `public/images/website/` — see [`public/images/website/README.md`](public/images/website/README.md)
  (products, containers, bulk oil, UCO collection, warehouse, delivery/operations, team — one flat folder)

Each README lists recommended filenames, dimensions, formats and max file sizes, and explains how the images get placed once uploaded (no code restructuring — placeholders swap out automatically).
